# 可视化拖拽组件

## 目标

主要流程（中途可能会补充或者调整顺序）

- [ ] 主页面结构：左侧可选组件列表、中间容器画布、右侧编辑组件定义好的属性
- [ ] 从菜单拖拽组件到容器；
- [ ] Block的选中状态；
- [ ] 容器内的组件可以拖拽移动位置；
- [ ] 命令队列以及对应的快捷键；
- [ ] 单选、多选；
- [ ] 设计好操作栏按钮：
    - [ ] 撤销、重做；
    - [ ] 导入、导出；
    - [ ] 置顶、置底；
    - [ ] 删除、清空；
- [ ] 拖拽贴边；
- [ ] 右键操作菜单；
- [ ] 组件可以设置预定义好的属性(props)；
- [ ] 拖拽调整宽高(height,width)；
- [ ] 组件绑定值(model)；
- [ ] 根据组件标识，通过作用域插槽自定义某个组件的行为(slotName);
- [ ] 完善可选组件列表:
    - [ ] 输入框：双向绑定值、调整宽度；
    - [ ] 按钮：类型、文字、大小尺寸、拖拽调整宽高；
    - [ ] 图片：自定义图片地址，拖拽调整图片宽高；
    - [ ] 下拉框：预定义选项值，双向绑定字段；
